💾 استمرارية البيانات (Database)

Railway يمسح الملفات عند كل نشر. البيانات الدائمة تُخزّن في PostgreSQL فقط. اضغط الزر أدناه لحفظ جميع بيانات المتاجر الحالية في قاعدة البيانات يدوياً.

كيف تضيف PostgreSQL في Railway؟
1. افتح مشروعك في Railway Dashboard
2. اضغط New ← Database ← Add PostgreSQL
3. في إعدادات التطبيق ← Variables ← تأكد أن DATABASE_URL موجودة
4. أعد تشغيل التطبيق — سيتصل بـ DB تلقائياً

آخر 200 حدث وارد من سلة (يُعاد ضبطه عند إعادة تشغيل الخادم)

🔐 التحقق من التوقيع (Signature)

سلة ترسل X-Salla-Signature مع كل webhook.

النظام يتحقق منه تلقائياً باستخدام HMAC-SHA256 مع متغير البيئة SALLA_WEBHOOK_SECRET.

إذا كان المتغير غير محدد، يتم قبول جميع الطلبات (وضع التطوير).
